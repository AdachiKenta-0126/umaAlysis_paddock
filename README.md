# umaAlysis_paddock
競馬には，レース前に競走馬の状態や調子を確認できる時間がある．

その時間を場所にちなんでパドックという．

これはパドック映像から画像処理による骨格検出を行うことで，

競走馬のポテンシャルをパドック映像から見出そうとしたものである．

骨格の[x,y]のcsvのファイルを入力とし，重回帰分析により順位の推定を行っている
